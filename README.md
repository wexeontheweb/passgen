# 🔐 Passgen

Lekki i bezpieczny generator haseł, który pozwala szybko tworzyć silne, trudne do złamania ciągi znaków.

---

## ✨ Funkcje

* **Customizacja długości:** Generuj hasła od 6 do 50 znaków.
* **Filtry znaków:** Wybieraj, czy hasło ma zawierać wielkie litery, małe litery, cyfry oraz znaki specjalne.
* **Bezpieczeństwo:** Wygenerowane hasła są generowane w twojej przeglądarce, więc nikt nie ma dostępu do twojego hasła oprócz **CIEBIE**
* **Copy-to-clipboard:** Szybkie kopiowanie wygenerowanego hasła jednym kliknięciem.
* **Wskaźnik siły:** Dynamiczna ocena, jak bezpieczne jest Twoje nowe hasło.
